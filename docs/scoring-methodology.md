# Scoring Methodology

> **Stub** — full content added in Phase 5.

This document describes how Tessera scores Claude Code Skills across six dimensions and three composite methodologies.

## Dimensions

- **Velocity** — commit momentum and acceleration
- **Adoption** — stars, forks, watchers
- **Freshness** — recency and activity
- **Documentation** — SKILL.md structure and completeness
- **Contributors** — unique contributor count
- **Code Quality** — presence of license, CI, tests, .gitignore, topics

## Composite Methodologies

| Methodology | Default? | Focus |
|-------------|----------|-------|
| Trending | Yes | Momentum signals |
| Popular | No | Ecosystem adoption |
| Well-Rounded | No | Balanced strength |
